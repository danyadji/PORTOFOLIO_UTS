Bagaimana Cara Deploy di Github?

Dengan menggunakan GitHub Pages, Anda dapat membuat situs web hanya dengan menerbitkan file HTML, JavaScript, dan CSS statis dari repositori GitHub. Di antara sekian banyak manfaat layanan hosting web GitHub, beberapa manfaat utamanya adalah betapa mudah dan sederhananya layanan ini digunakan. Dengan mempertimbangkan semua itu, mari kita bahas lebih rinci cara menghosting situs web di GitHub sendiri.

Coba Sekarang!

Step-step deploy di Github

Persiapkan Repositori di GitHub

    1. Buat Repositori Baru:
    Buka GitHub dan login ke akun Anda. Klik New Repository. Beri nama repositori sesuai dengan proyek Anda. Set repositori menjadi public agar bisa diakses melalui GitHub Pages. Klik Create repository.

    2. Upload Proyek ke Repositori:
    Jika proyek sudah ada di lokal, buka terminal atau command prompt, lalu navigasi ke folder proyek Anda. Inisialisasi Git dan tambahkan repositori GitHub sebagai remote: 

git init git remote add origin 
https://github.com/username/repository-name.git

    Tambahkan semua file dan lakukan commit pertama: 

git add .
git commit -m "Initial commit"

    Kirim file ke GitHub:

git push -u origin main

Mengaktifkan GitHub Pages

    1. Buka Pengaturan Repositori:
    Masuk ke repositori Anda di GitHub. Klik tab Settings.

    2. Aktifkan GitHub Pages:
    Gulir ke bawah hingga menemukan bagian GitHub Pages.
    Di bagian Source, pilih branch main dan pilih folder root (biasanya /).
    Klik Save untuk menyimpan pengaturan.

    3. Tunggu GitHub Memproses Deploy:
    Setelah diaktifkan, GitHub akan otomatis memproses deploy dan membuat halaman untuk proyek Anda.
    URL proyek akan tersedia dalam format: https://username.github.io/repository-name
