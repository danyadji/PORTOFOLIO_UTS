Apa itu Canvas di HTML?

Elemen HTML <canvas> digunakan untuk menggambar grafik, dengan cepat, melalui skrip (biasanya JavaScript). Elemen <canvas> hanyalah wadah untuk grafis. Anda harus menggunakan skrip untuk benar-benar menggambar grafiknya. Canvas memiliki beberapa metode untuk menggambar jalur, kotak, lingkaran, teks, dan menambahkan gambar.

Contoh Canvas

    Di HTML, Syntax canvas seperti berikut

<canvas id="myCanvas" width="200" height="100"></canvas>

HTML Canvas gambar

    drawImage(image, dx, dy)


<script>
  const canvas = document.getElementById("myCanvas");
  const ctx = canvas.getContext("2d");
  const image = document.getElementById("scream");
                    
  image.addEventListener("load", (e) => {
  ctx.drawImage(image, 10, 10);
  });
</script>

drawImage(image, dx, dy, dwidth, dheight)

    drawImage(image, dx, dy, dwidth, dheight) sintaks memposisikan gambar di kanvas, dan menentukan lebar dan tinggi gambar di kanvas. 


<script>
  const canvas = document.getElementById("myCanvas");
  const ctx = canvas.getContext("2d");
  const image = document.getElementById("scream");
                    
  image.addEventListener("load", (e) => {
    ctx.drawImage(image, 10, 10, 80, 80);
  });
</script>
