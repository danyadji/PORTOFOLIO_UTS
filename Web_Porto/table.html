Basic Table di HTML

Tabel HTML memungkinkan pengembang mengatur data dalam baris dan kolom untuk tampilan yang terstruktur dan mudah dibaca. Dengan tabel, informasi kompleks dapat disajikan secara sistematis, memudahkan analisis dan perbandingan. Selain itu, penggunaan pengaturan dan CSS dapat mempercantik tampilan tabel sesuai kebutuhan desain.

Coba Sekarang!

Tag-Tag Table di HTML

<table>

Defines a table

<tr>

Defines a row in a table

<td>

Defines a cell in a table

<caption>

Defines a table caption

<colgroup>

Specifies a group of one or more columns in a table for formatting/p>

<col>

Specifies column properties for each column within a <colgroup> element

<thead>

Groups the header content in a table

<tbody>

Groups the body content in a table

<tfoot>

Groups the footer content in a table

Cara Membuat!

Untuk membuat tabel dasar dalam HTML, Anda dapat menggunakan elemen <table> yang berfungsi sebagai wadah utama. Baris dalam tabel dibuat menggunakan elemen <tr> (table row), sedangkan sel-sel dalam baris tersebut menggunakan elemen <td> (table data) untuk isi data dan <th> (table header) untuk judul kolom. Agar tabel terlihat lebih rapi, Anda bisa menambahkan atribut seperti border untuk memberi garis tepi pada tabel, sehingga memudahkan dalam pembacaan data yang ditampilkan.
